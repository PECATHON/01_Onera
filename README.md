# RealWorld Example App

This repository contains both frontend and backend implementations of the RealWorld example application.

## Structure

- `frontend/` - React + Redux frontend application
- `backend/` - Node.js + Express + Prisma backend API
- `demo-video.mp4` - Application demonstration video

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Git

### Setup

1. Clone this repository
2. Setup backend (see backend/README.md)
3. Setup frontend (see frontend/README.md)

## Demo Video

Check out `demo-video.mp4` for a demonstration of the application features.
